# Provider Logos

Place provider SVG logos here using lowercase provider IDs as filenames.

Examples:
- `openai.svg`
- `anthropic.svg`
- `google_cloud.svg`
- `together_ai.svg`

The app will automatically load `/logos/providers/<provider_id>.svg`.
If a logo file is missing, a generated fallback badge is shown.

Licensing:
- Use only official brand assets with permitted usage.
- Do not commit assets with unclear redistribution rights.
